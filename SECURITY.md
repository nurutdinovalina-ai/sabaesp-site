# Security Policy

## Reporting a Vulnerability

SABA ESP takes security seriously.

If you discover a potential security vulnerability related to this website
or its infrastructure, please report it responsibly by email.

**Contact:**  
info@sabaesp.com

Please include:
- a description of the issue  
- steps to reproduce (if applicable)  
- any relevant screenshots or logs  

We will review all reports and respond as quickly as possible.
